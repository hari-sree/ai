# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Purpose

This repo is a home for AI experiments, projects, and reusable assets. It serves two roles:

1. **AI projects** — a workspace to build and iterate on AI-related tools, scripts, and apps
2. **Templates** — a git-persisted library of reusable files (e.g. global `CLAUDE.md`, markdown configs, prompts) that can be pulled into new projects

## Structure

- `templates/` — reusable file templates (markdown, config, prompts, etc.)
